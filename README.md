# Task Manager

A small responsive task manager web app with semantic HTML, modern styling, and accessible interactions.

## Features

- Add new tasks
- Mark tasks complete
- Delete tasks
- Filter tasks by all, pending, or completed
- Dark mode toggle with theme persistence

## Files

- `index.html` — main page markup
- `styles.css` — responsive styles and dark mode themes
- `script.js` — task interactions, filtering, validation, and theme toggle

## Usage

Open `index.html` in a browser to use the task manager. Tasks are managed in-memory and can be added, completed, filtered, and deleted.

## Notes

This project is designed for a simple frontend demo and does not include server-side storage.
